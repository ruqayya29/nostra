# Nostra 👗

A responsive fashion e-commerce website built with HTML, CSS, and JavaScript.

## Pages

- **Home** — Hero image carousel, brand showcase, featured collections, new arrivals, most wanted, and special offer section
- **Collections** — Filter products by occasion, color, and arrival type with a live search bar
- **Contact** — Contact form for customer inquiries

## Features

- Responsive design for both desktop and mobile
- Slide-in mobile navigation menu
- Promotional popup banner with close button
- Product filtering by occasion, color, and arrival
- Horizontal scroll for Most Wanted section
- Newsletter subscription section

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- Vanilla JavaScript
- Google Fonts — Bricolage Grotesque
- Font Awesome Icons
